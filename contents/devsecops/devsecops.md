O DevSecOps introduz o conceito de segurança  como código(security as code)

- Code Review
- Modelagem de ameaças
- Teste de invasão

Encoraja a automação para resiliência.

Pŕe produção para produção:

- Plan
- Code
- Build
- Test
- Release
- Deploy
- Monitor
- Respond
